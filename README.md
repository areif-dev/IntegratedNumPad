# Integrated Numpad

## Intro
Remaps numpad keys to a dedicated cluster on a keyboard. Useful for TKL or other compact boards. The remapping happens as follows:

- b: 0
- n: 1
- m: 2
- ,: 3
- h: 4
- j: 5
- k: 6
- y: 7
- u: 8
- i: 9
